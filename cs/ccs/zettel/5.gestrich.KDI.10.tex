\section{Pumping Lemma}
Assume there is a number $ n \in \N  $ so that for all words $ x \in L_3, \left| x \right| \geq n $ the word can be written as $ uvw $ so that $ \left| v \right| \geq 1, \left| uv \right| \leq n, \forall i \in N_0: uv^iw \in L_3 $.
Take the word $ x \coloneqq e^nd^{2n} $, so that $ \left| x \right| \geq n $,
so that $ x = uv^iw $. Since $ \left| uv \right| \leq n $ $ v $ has to contain only $ e $'s because the first $ n $ symbols are $ e $'s.
And because $ \left| v \right| \geq 1 $ it means $ v $ contains at least one $ e $.
That means one could loop $ v $ once more, so $ e^{n+\left| v \right| }d^{2n} $ would be also in $ L_3 $, but that is in contradiction to the definition, that means $ L_3 $ is not regular.
